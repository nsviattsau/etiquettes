Impression d'étiquettes DataMatrix

Format style ZPL : [Code] [DataMatrix] ou [DataMatrix] [Code], selon la position choisie.
Une ligne = une étiquette.
Si le texte et le DataMatrix sont identiques : écrivez seulement le code.
Si le texte et le DataMatrix sont différents : utilisez texte ; datamatrix.

Aucune étiquette générée.
Les étiquettes apparaîtront ici.
